Frequently Asked Questions

Can't find the answer you're looking for?
Reach out to Upscayl Support.

What is Upscayl Cloud?

Upscayl Cloud is a cloud-powered version of Upscayl that allows you to upscale images without requiring powerful hardware on your device.

How is it different from Upscayl Desktop?

Unlike desktop version, Upscayl Cloud runs entirely in your browser and uses cloud computing power, meaning no local GPU required.

How do I use Upscayl Cloud Beta?

Simply sign up for beta access, upload your images, select your preferred settings, and let our cloud infrastructure handle the processing.

How much does Upscayl Cloud Beta cost?

During the beta period, Upscayl Cloud is free to use. We'll announce pricing details before the official launch.

What models are available on Upscayl Cloud Beta?

We support Real-ESRGAN, ESRGAN, and multiple other AI models optimized for different types of images.

Do you plan an API for Upscayl Cloud Beta?

Yes! API access is planned for commercial users and will be available after the beta period.

Do we get API Keys for Upscayl Cloud Beta?

API keys will be provided to eligible users during the commercial launch phase.

How do I access my upscaled images?

All upscaled images are available for immediate download and are stored in your account for 30 days.
