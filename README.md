# Upscayl Design - Shadcn Inspired UI

A beautiful, modern UI design inspired by shadcn's design language. This project features a clean, aesthetic interface with smooth animations and thoughtful interactions.

## Features

- **Shadcn Design Language**: Follows shadcn's color palette, typography, and spacing system
- **Modern Aesthetics**: Clean, minimal design with subtle shadows and borders
- **Smooth Animations**: Thoughtful transitions and hover effects throughout
- **Responsive Design**: Fully responsive layout that works on all devices
- **Accessible**: Focus states and semantic HTML for better accessibility

## Design Principles

- **Subtle Borders**: Using hsl color variables for consistent border colors
- **Muted Color Palette**: Soft, professional color scheme
- **Smooth Transitions**: Cubic-bezier easing functions for natural motion
- **Visual Hierarchy**: Clear typography scale and spacing system
- **Interactive Elements**: Hover states and micro-interactions enhance UX

## Getting Started

Simply open `index.html` in your browser to view the design.

## Components

- **Header**: Sticky navigation with backdrop blur
- **Hero Section**: Gradient title with call-to-action buttons
- **Feature Cards**: Animated cards with hover effects
- **Upload Section**: Interactive file upload area with options
- **Stats Section**: Display key metrics
- **Footer**: Clean footer with links

## Color System

The design uses CSS custom properties based on shadcn's color system:
- Primary colors for accents and CTAs
- Muted colors for secondary text
- Border colors for subtle separations
- Background variations for depth

## Browser Support

Modern browsers with CSS custom properties and backdrop-filter support.
